# Dataset Documentation

## SmartCare Health Monitoring Dataset

This folder contains the data resources used for the SmartCare Health Monitoring Analytics project.

The project analyses smartwatch sensor data to identify:

- Health behaviour patterns
- Activity trends
- Sleep characteristics
- Potential health risk indicators
- Preventive healthcare insights

---

## Dataset Overview

The dataset represents wearable device measurements collected from users, including health and activity-related attributes.

Typical data categories include:

| Category | Description |
|---|---|
| User Information | User-level identifiers and demographic attributes |
| Activity Metrics | Physical activity and movement measurements |
| Sleep Metrics | Sleep duration and sleep behaviour indicators |
| Health Metrics | Health-related measurements collected through wearable devices |
| Risk Indicators | Derived features used for analytical assessment |

---

## Data Processing

The data analysis pipeline includes:

1. Data Loading
2. Data Quality Assessment
3. Missing Value Handling
4. Data Cleaning
5. Exploratory Data Analysis
6. Health Pattern Identification
