# Python Source Code

This folder contains the core Python scripts developed for the SmartCare Health Monitoring Analytics project.

## Files

## smartcare_analysis.py

Main analytics pipeline responsible for:

- Loading smartwatch health data
- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical analysis
- Identifying health trends and risk indicators
- Generating analytical insights


## smartcare_dashboard.py

Interactive dashboard application responsible for:

- Visualising healthcare insights
- Displaying health trends
- Presenting activity and risk indicators
- Supporting data-driven healthcare decisions
